## Run Locally

Clone the project

```bash
  git clone https://github.com/Sr-vZ/PyAudioPlayer.git
```

Go to the project directory

```bash
  cd PyAudioPlayer
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python app.py
```
